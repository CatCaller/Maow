# Maow
Funny program for trolling

## Features
- Cat windows that bounce around the screen
- GIF animation
- Random popup messages with cute icon
- Tunnel zoom
- Background audio playback (Caramelldansen)
- Killswitch by typing "maow"
- Speed up gif movement when mouse is moved
- Wallpaper replacing
- Killing WallpaperEngine
- Multi-monitor support
- Collision stuff
- Jittery mouse
- Bad config system
- Broken song ended detection
